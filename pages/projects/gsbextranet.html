GSB Admin

GSB Admin est une application web qui permet de gérer le site extranet GSB, développée en PHP. Elle permet de mettre en maintenance le site, ajouter des produits, voir des statistiques sur les produits...

L'application dispose de plusieurs écrans, dont les suivants :

  • Page d'accueil pour l'administrateur
  • Page pour mettre en maintenance
  • Page d'accueil pour les chef de produit
  • Page de gestion des produits
  • Page d'ajout de produit
  • Page de statistiques
  • Page de statistiques des produits les plus visités
