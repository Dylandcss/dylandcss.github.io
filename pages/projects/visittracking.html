VisitTracking

Ce projet est une application Windows développée pour une entreprise fictive GSB. Créée en C# avec le framework .Net et Windows Forms, elle permet de suivre les visites des commerciaux chez les clients. Elle permet également grâce à un système de CRUD de gérer les clients, les commerciaux et les visites (visualiser, créer, modifier, supprimer).

Ce projet m'a permi de découvrir en profondeur le CRUD et de me familiariser avec les concepts de la programmation orientée objet. J'ai appris à créer des classes, des formulaires et des vues. J'ai également appris à créer une API en PHP et à l'appeler en C# pour interagir avec la base de données.

J'ai utilisé Git pour versionner mon code et GitHub pour héberger mon application. J'ai utilisé Visual studio Community 2022 pour développer mon application.
