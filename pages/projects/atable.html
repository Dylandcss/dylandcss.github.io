À table

Ce projet est une application mobile de gestion de recettes de cuisine. Elle permet de trouver des recettes à partir des ingrédients que l'on dispose. Elle utilise une API privée, développée de A à Z.

Ce projet m'a permi de découvrir le développement mobile avec Flutter. Avec l'utilisation d'une API. Les widgets de Flutter m'ont permis de créer une interface utilisateur intuitive et agréable.

J'ai utilisé Git pour versionner mon code et GitHub pour héberger mon application. J'ai utilisé IntelliJ IDEA pour développer mon application.
