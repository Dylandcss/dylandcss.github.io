Contexte GSB

GSB est un laboratoire (fictif) qui intervient dans l'industrie pharmaceutique, le laboratoire souhaite moderniser son activité de visite médicale. Notamment en uniformisant la gestion du suivi des visites.

Dans le cadre de cette modernisation, plusieurs applications ont été développés. Parmi elles, nous pouvons retrouver les suivantes :

  • GSB Admin - Application Web
  • VisiTTracking - Application Client lourd
  • GSB Véhicule - Application Web
  • MV - Application Mobile
