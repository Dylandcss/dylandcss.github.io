Portfolio

Ce projet est mon portfolio personnel. Il est réalisé en HTML, CSS et JavaScript. Il est hébergé sur GitHub Pages. Il est responsive et s'adapte à tous les écrans. Il est composé de plusieurs pages :

  • Accueil
  • Projets
  • Veille

Ce projet m'a permis de mettre en pratique mes compétences en développement web. J'ai pu travailler sur la structure du site, le design, le responsive, l'accessibilité et l'optimisation du site.

J'ai utilisé Git pour versionner mon code et GitHub pour héberger mon site. J'ai également utilisé Visual Studio Code pour coder mon site.

Ce projet m'a permis de mettre en pratique mes compétences en développement web et de montrer mes compétences à d'autres personnes.
